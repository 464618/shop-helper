GitHub Gist 配置

Token 只保存在当前浏览器，不会导出、不会上传到 Gist。不要把 token 发给任何人，建议只给 Gist 写权限。

普通同步

普通同步会先拉取云端数据，再按 id 合并本地和云端，保留 updatedAt 较新的版本。删除采用软删除。

强制操作

强制上传会用当前设备数据覆盖云端；强制拉取会用云端覆盖当前设备。请谨慎使用。

本地备份
